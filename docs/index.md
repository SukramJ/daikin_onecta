# daikin_onecta

Home Assistant custom integration for Daikin devices via the Daikin
Onecta cloud API.

This site collects the engineering documentation for the integration:
architecture, security, release process, and the architecture decision
records.

User-facing documentation lives in the project [README] and the
[GitHub repository][repo].
